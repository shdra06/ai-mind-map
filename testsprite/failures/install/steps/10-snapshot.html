Install in One Command

Auto-detects and configures your AI agent. No config files. No manual setup.

1
2
3
4

Choose Your Agent

Select the AI agent you're using. AI Mind Map supports all of them.

Run the Installer

One command auto-detects Cursor and configures everything.

npx ai-mind-map install

Note Works on macOS, Linux, and Windows. Requires Node.js ≥ 18.

.cursor/mcp.json
{
  "mcpServers": {
    "ai-mind-map": {
      "command": "npx",
      "args": [
        "-y",
        "ai-mind-map"
      ]
    }
  }
}

Verify the Connection

Restart your agent, then ask it:

Call mindmap_status to verify the connection

Expected response:

{
  "status": "connected",
  "version": "1.21.2",
  "tools": 50,
  "project": "your-project",
  "indexed": true,
  "symbols": 1247,
  "edges": 3891
}

You're Done!

Your agent now has persistent codebase memory.
